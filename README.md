# Lyxel

> Interfaz gráfica para **[Scrcpy](https://github.com/Genymobile/scrcpy)** con comandos ADB personalizados y herramientas de optimización para Android.
>
> [![GitHub release](https://img.shields.io/github/v/release/sudoNks/Lyxel?label=última%20versión)](https://github.com/sudoNks/Lyxel/releases/latest)
> [![GitHub all releases](https://img.shields.io/github/downloads/sudoNks/Lyxel/total?label=descargas)](https://github.com/sudoNks/Lyxel/releases)
>
> > ⚠️ **Este proyecto utiliza [Scrcpy](https://github.com/Genymobile/scrcpy)** como motor de espejo/control. Lyxel es una GUI independiente que lo envuelve — no es un fork ni modifica el código de Scrcpy.
> >
> > ---
> >
> > ## ¿Qué es Lyxel?
> >
> > Lyxel (anteriormente **MobiladorSteX**) es una interfaz gráfica para Windows que permite usar Scrcpy de forma visual, sin necesidad de usar la terminal. Incluye además comandos ADB propios y, desde la v1.3.0, herramientas de optimización para dispositivos Android.
> >
> > ### Funcionalidades principales
> > - GUI completa para Scrcpy (resolución, bitrate, orientación, etc.)
> > - - Comandos ADB adicionales propios
> >   - - **v1.3.0+**: Módulo de optimización para Android vía ADB

> > ## 📋 Requisitos
> >
> > - Windows 10/11 (x64)
> > - - [Scrcpy](https://github.com/Genymobile/scrcpy) instalado y en el PATH
> >   - - ADB instalado (incluido con Scrcpy o Android Platform Tools)
> >     - - Dispositivo Android con depuración USB habilitada
