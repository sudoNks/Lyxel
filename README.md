# ✨ Lyxel

> **Antes conocido como MobiladorSteX**
>
> Lyxel es una interfaz gráfica (GUI) para [Scrcpy](https://github.com/Genymobile/scrcpy) que simplifica el control de dispositivos Android desde PC, con comandos ADB personalizados integrados. A partir de la versión v1.3.0, incluye además un módulo de optimización Android vía ADB.
>
> > ⚠️ **Lyxel NO es un fork de Scrcpy.** Es un proyecto independiente que usa Scrcpy como herramienta subyacente. Scrcpy es desarrollado por [Genymobile](https://github.com/Genymobile/scrcpy) bajo licencia Apache 2.0.

> > ## 📋 Requisitos
> >
> > - Windows 10/11 (x64)
> > - - [Scrcpy](https://github.com/Genymobile/scrcpy) instalado
> >   - - ADB disponible en el sistema
> >     - - Depuración USB habilitada en el dispositivo Android
