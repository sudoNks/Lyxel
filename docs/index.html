Funciones

Todo en uno.

LyXel no es un fork de Scrcpy. Es un proyecto independiente que usa Scrcpy como base y agrega interfaz propia, optimizaciones ADB y nuevas funcionalidades.

Transmisión de pantalla
Video y audio desde tu Android a Windows. FPS, bitrate, resolución y codec configurables.
Optimización ADB
Ajustes para Samsung, Xiaomi, Pixel, Huawei y Free Fire con un solo toggle. Sin comandos.
Controles configurables
Activa o desactiva teclado, mouse y gamepad. Modo UHID para máxima compatibilidad.
Conexión WiFi
Conecta de forma inalámbrica. Guarda la sesión y reconecta sin configurar de nuevo.
Perfiles guardados
7 perfiles preconfigurados. Crea, exporta e importa los tuyos. Cambia configuración en un clic.
Modo de renderizado
Direct3D, OpenGL, OpenGLES2 o Software. Filtrado automático según tu arquitectura.

Descargar

Elige tu versión.

Scrcpy y ADB incluidos. No necesitas instalar nada adicional.

Windows 10/11 — Recomendado 64 bits (x64) Para la mayoría de PCs modernos. Rendimiento óptimo y compatibilidad completa. Descargar .exe
Windows 10 — Compatibilidad 32 bits (x86) Para equipos más antiguos. Activa el Modo Compatibilidad dentro de la app tras instalar. Descargar .exe

Requisitos

Lo que necesitas.

Solo tu PC con Windows y tu teléfono Android. Todo lo demás ya viene incluido.

PC — Windows
  • Windows 10 o Windows 11
  • Arquitectura x86 (32 bits) o x64 (64 bits)
  • Scrcpy y ADB incluidos — no requieren instalación adicional
  • Puerto USB disponible para la primera conexión
Teléfono — Android
  • Android 13 o superior (recomendado Android 11 mínimo)
  • Depuración USB habilitada en Opciones de desarrollador
  • En Xiaomi: activar "Depuración USB (modo seguridad)"
  • Cable USB para configuración inicial (WiFi opcional después)
