🤖 ¿Qué es Lyxel?

Lyxel (antes MobiladorSteX) es una interfaz gráfica independiente para Scrcpy que facilita el control de dispositivos Android desde tu PC. En lugar de tener que armar scripts BAT o memorizar comandos ADB, Lyxel te da una interfaz clara con las opciones más útiles a un clic.

A partir de la versión v1.3.0, incluye además un módulo de optimización Android que aplica configuraciones vía ADB para mejorar el rendimiento del dispositivo.

⚠️ Nota: Lyxel no es un fork de Scrcpy. Es un proyecto independiente. Scrcpy es desarrollado por Genymobile bajo licencia Apache 2.0.

📋 Requisitos

💻

Sistema Operativo

Windows 10 (x86 o x64) o Windows 11 (x64).

📱

Dispositivo

Un dispositivo Android con la Depuración USB habilitada.

🔔

Todo Incluido

Scrcpy y ADB ya vienen integrados. No necesitas instalar nada extra.
